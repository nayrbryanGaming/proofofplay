# Proof of Play - Final Setup Checklist

**Before Demo:**
1. [ ] Install Node.js (v18+). Run `node -v` to verify.
2. [ ] Open Terminal: `cd proof_of_play/app`
3. [ ] Run `yarn install`
4. [ ] Run `yarn dev`
5. [ ] Open Browser: `http://localhost:3000`

**During Demo:**
1. [ ] Open `DEMO_SCRIPT.md` (Keep script handy).
2. [ ] Connect Phantom Wallet (Devnet).
3. [ ] Click `Init Player` -> Approve.
4. [ ] Click `Explore` -> Approve.
5. [ ] Click `Fight` -> Approve.
6. [ ] Click `Claim` -> Approve.
7. [ ] Show Visualizer (Optional).
8. [ ] Show Explorer Link.

**Troubleshooting:**
- If `yarn install` fails: Delete `node_modules` and `yarn.lock`, then run `yarn install` again.
- If `yarn dev` fails: Check `node -v` (Must be > v18).
